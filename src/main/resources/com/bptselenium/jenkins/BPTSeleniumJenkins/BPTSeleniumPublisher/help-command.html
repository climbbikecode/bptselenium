To generate a command for a suite:
  1. Go to "Test Lab" on the BPTSelenium website
  2. Double-click the suite you want to run
  3. Click "Run tests" or "Run suite" depending on which you want to do
  4. Configure parameters in the "Run configuration" box
  5. Click "Show command"
  6. Copy the command from the website to the command box for the BPTSelenium plugin in Jenkins
